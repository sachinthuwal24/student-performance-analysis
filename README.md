# Student Performance Analysis

## Project Overview
This project analyzes student performance using Python.

## Features
- Calculate average marks
- Find topper
- Find class average
- Visualize marks using graphs

##  Technologies Used
- Python
- Pandas
- Matplotlib

##  Files
- students.csv → dataset
- final_report.csv → processed data
- student_performance_analysis.ipynb → main code

## Output
- Bar graph visualization
- Topper detection
- Class performance analysis
